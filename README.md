# Database Optimization Tool

Static SEO-first website for `databaseoptimizationtool.com`.

## Positioning

Database Optimization Tool helps small businesses, local shops, and small engineering teams start database performance work with evidence instead of risky production changes.

PostgreSQL is available first. MySQL and SQL Server pages are roadmap surfaces until collectors and findings are implemented.

## Commands

```bash
npm run check
npx wrangler pages deploy public --project-name database-optimization-tool
```

## Deployment

Cloudflare Workers serves the `public/` directory through Workers static assets.
